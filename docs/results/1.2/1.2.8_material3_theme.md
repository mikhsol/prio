## Theme Overview

The Jeeves theme implements Material Design 3 with custom brand colors and Eisenhower quadrant color tokens per [1.1.13 Component Specifications](../1.1/1.1.13_component_specifications.md).

## Theme Features

### Dynamic Color Support

```kotlin
val colorScheme = when {
    dynamicColor && Build.VERSION.SDK_INT >= Build.VERSION_CODES.S -> {
        if (darkTheme) dynamicDarkColorScheme(context) 
        else dynamicLightColorScheme(context)
    }
    darkTheme -> DarkColorScheme
    else -> LightColorScheme
}
```

Android 12+ devices use Material You dynamic theming, falling back to brand colors on older devices.

### Edge-to-Edge Support

```kotlin
SideEffect {
    val window = (view.context as Activity).window
    window.statusBarColor = colorScheme.surface.toArgb()
    WindowCompat.getInsetsController(window, view)
        .isAppearanceLightStatusBars = !darkTheme
}
```

## Usage

```kotlin
@Composable
fun MyScreen() {
    JeevesTheme {
        // Access theme colors
        val colors = MaterialTheme.colorScheme
        val typography = MaterialTheme.typography
        
        Text(
            text = "Do First",
            color = QuadrantColors.doFirst,
            style = typography.titleMedium
        )
    }
}
```

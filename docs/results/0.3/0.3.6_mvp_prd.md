## 1. Executive Summary

### 1.1 Product Vision

Prio is a privacy-first, AI-powered personal assistant that runs primarily on-device to help you take control of your life. Built around the Eisenhower Matrix philosophy, Prio transforms how you manage tasks, goals, and time—anticipating your needs while keeping your data secure on your smartphone.

### 1.2 MVP Value Proposition

**"Your Private Productivity AI"** — The first Android productivity assistant with on-device AI that automatically prioritizes tasks using the Eisenhower Matrix while keeping all data local.

### 1.3 Key Differentiators

| Differentiator | Description | Proof Point |
|----------------|-------------|-------------|
| **On-Device AI** | AI runs locally on the phone | Phi-3-mini via llama.cpp; no cloud required |
| **Automatic Prioritization** | AI assigns Eisenhower quadrants | 80%+ accuracy on classification |
| **Goal-Task Integration** | Connect daily tasks to long-term goals | Unique in productivity app market |
| **Daily Briefings** | AI-generated morning/evening summaries | Personalized based on tasks, goals, calendar |
| **Privacy-First** | All data stays on device | No account required; no cloud sync in MVP |

### 1.4 Target Launch

- **Beta**: Week 14 (14 weeks from start)
- **Public Launch**: Week 16 (16 weeks from start)
- **Target Downloads (Month 1)**: 10,000
- **Target Reviews**: 4.5+ star rating

## 3. Feature Specification

### 3.1 Feature Summary

| Feature Area | P0 (Must-Have) | P1 (Should-Have) | Total Stories |
|--------------|----------------|------------------|---------------|
| Task Management | 7 | 3 | 10 |
| Goals & Progress | 4 | 2 | 6 |
| Calendar & Briefings | 3 | 3 | 6 |
| **Total** | **14** | **8** | **22** |

### 3.2 Task Management

#### P0 Features (Must-Have)

**TM-001: Quick Task Capture**
- FAB visible on all main screens
- Voice and text input
- Task saved in <5 seconds

**TM-002: Natural Language Parsing**
- Parse task title, due date, priority from natural language
- "Call mom tomorrow at 5pm" → Title: Call mom, Due: Tomorrow 5PM
- Works offline with on-device AI

**TM-003: AI Eisenhower Classification**
- Automatically assign Q1/Q2/Q3/Q4 to every task
- Classification within 2 seconds
- 80%+ accuracy target
- User can override with one tap

**TM-004: Task List with Priority Indicators**
- Color-coded quadrant badges (Q1=Red, Q2=Blue, Q3=Amber, Q4=Gray)
- Sorted by priority
- Overdue tasks highlighted

**TM-005: Deadline-Based Urgency**
- Urgency increases as deadline approaches
- Contributes to Eisenhower classification
- Configurable notification intervals

**TM-006: Complete, Edit, Delete Tasks**
- Swipe to complete/delete
- Bottom sheet for editing
- Undo support

**TM-010: Override AI Classification**
- Tap quadrant badge to change
- Store overrides for model improvement

#### P1 Features (Should-Have)

**TM-007: Subtasks/Checklists** - Break tasks into sub-items

**TM-008: Recurring Tasks** - Daily, weekly, monthly patterns

**TM-009: Smart Reminders** - Context-aware notifications

### 3.3 Goals & Progress

#### P0 Features (Must-Have)

**GL-001: Create Goal with AI**
- Natural language goal description
- AI suggests SMART improvements
- Category selection (Career, Health, Personal, etc.)

**GL-002: Progress Visualization**
- Progress bar on each goal
- Color status (green=on track, yellow=behind, red=at risk)
- Celebration animation at 100%

**GL-003: Link Tasks to Goals**
- Connect tasks to goals
- Completing tasks updates goal progress
- AI suggests goal links

**GL-005: Goal Dashboard**
- Overview with summary stats
- Sort by status/deadline
- Filter by category

#### P1 Features (Should-Have)

**GL-004: Goal Milestones** - Intermediate checkpoints

**GL-006: Goal Analytics** - Progress charts and insights

### 3.4 Calendar & Briefings

#### P0 Features (Must-Have)

**CB-001: Morning Daily Briefing**
- Personalized morning summary
- Top 3 priority tasks, calendar overview, goal spotlight
- AI-generated insight
- Push notification at configured time

**CB-002: Calendar Read Integration**
- Connect to device calendars
- Display events in Prio
- Read-only in MVP

**CB-005: Calendar Day View**
- Timeline with events and tasks
- Combined schedule view
- Swipe to navigate days

#### P1 Features (Should-Have)

**CB-003: Evening Summary** - Review accomplishments, plan tomorrow

**CB-004: Meeting Notes** - Take notes, extract action items

**CB-006: Smart Insights** - AI-powered productivity advice

### 3.5 On-Device AI

**AI-001: Model Integration**
- Phi-3-mini-4k-instruct model
- llama.cpp via JNI
- 2-3 second inference time

**AI-002: Model Download**
- First-run download flow
- ~2.5 GB model size
- WiFi recommended warning
- Progress indicator

**AI-003: Rule-Based Fallback**
- For devices that can't run LLM
- Regex + keyword based classification
- 75% accuracy baseline

### 3.6 Foundation

**FND-001: Onboarding**
- 5-screen welcome flow
- Model download
- Permissions (calendar, notifications)
- First task creation

**FND-002: Offline Storage**
- Room database
- All data local
- No required network

**FND-003: Settings**
- Briefing times
- Notification preferences
- Theme (dark/light/system)
- AI model management

|-------------|---------------|

| Target SDK | API 34 (Android 14) |

| Component | Technology |

## 6. Design Requirements

### 6.1 Design Principles

1. **Clarity First**: Priority and status visible at a glance
2. **Speed of Capture**: Task creation in <5 seconds
3. **Trust Building**: Show AI reasoning, allow overrides
4. **Privacy Visible**: Clear indicators that data is local
5. **Calm Interface**: Reduce visual noise; focus on action

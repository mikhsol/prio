## Component Inventory

### 1. QuadrantBadge.kt

**Purpose**: Displays Eisenhower Matrix quadrant indicator

**Contents**:
- `Quadrant` enum - All 4 quadrants with colors, emojis, descriptions
- `BadgeSize` enum - COMPACT (24dp), STANDARD (28dp), LARGE (48dp)
- `QuadrantBadge` composable - Main component with 3 size variants

**Key Features**:
- Theme-aware colors for light/dark mode
- Optional tap handler for priority selection
- Full accessibility with content descriptions
- Animated transitions when tapped

### 2. TaskCard.kt

**Purpose**: Display individual task with priority and completion status

**Contents**:
- `TaskCardData` - UI model for task display
- `TaskCard` composable - Main card component

**Key Features**:
- 72dp minimum height per spec
- Quadrant badge integration
- Checkbox with 48dp touch target
- Overdue state with 3dp red left border
- Completed state with strikethrough and 60% opacity
- Selected state with primary color tint
- Metadata row (due date, goal linking)

### 3. GoalCard.kt

**Purpose**: Display goal with progress tracking

**Contents**:
- `GoalCategory` enum - 6 categories with emojis
- `GoalStatus` enum - ON_TRACK, SLIGHTLY_BEHIND, AT_RISK
- `ProgressVariant` enum - LINEAR, CIRCULAR
- `GoalCardData` - UI model for goal display
- `GoalCard` composable - Main card component
- `LinearProgress` composable - Animated progress bar
- `CircularProgress` composable - Animated circular progress

**Key Features**:
- Category icon and label
- Animated progress (600ms tween)
- Status color coding per spec
- Metadata: target date, milestones, linked tasks
- Linear and circular progress variants

### 4. MeetingCard.kt

- `MeetingCardData` - UI model for meeting display

**Key Features**:

- Vertical divider matching calendar color
- "NOW" indicator for ongoing meetings
- Attendees count
- Action items count
- Notes indicator

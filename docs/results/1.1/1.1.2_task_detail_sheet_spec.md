## Purpose

The Task Detail sheet is a bottom sheet that allows users to view and edit all task properties. It provides full editing capabilities while keeping the user in context of the task list. The sheet also displays AI classification reasoning to build trust.

**Design Priority**: Implements one-tap quadrant override (per persona validationâ€”users need to correct AI when wrong).

## AI Integration Points

### Classification Display

- AI explanation visible in half-expanded view
- Builds trust by showing reasoning
- User can override with one tap

### Goal Suggestion

- AI analyzes task content
- Suggests matching goal if confidence >70%
- Shows at bottom of goal selector
- User can accept or ignore

### Learning from Overrides

When user changes quadrant:
1. Store: original AI classification, user override
2. Track pattern: "User moves 'report' tasks from DELEGATE to DO"
3. Future improvement: Adjust classification for similar tasks

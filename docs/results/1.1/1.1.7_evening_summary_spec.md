## Close Day Flow

When user taps "Close Day & Plan Tomorrow":

1. **Apply task decisions**:
   - Move selected tasks to tomorrow
   - Reschedule as specified
   - Mark dropped tasks as Q4

2. **Animation**:
   - Checkmark scales up
   - Brief confetti burst
   - Haptic success feedback

3. **Transition**:
   - Summary card collapses
   - Dashboard shows "Day closed âœ“"
   - Ready for tomorrow's briefing

## Purpose

The Task List screen is the primary view for managing tasks. It displays all active tasks sorted by Eisenhower priority, with clear visual indicators for each quadrant. This is where users spend most of their time interacting with tasks.

**Design Priority**: This implements the core Eisenhower-powered prioritization that addresses Alex's #1 pain point: "50 tasks, don't know where to start."

---

## Entry Points

| From | Trigger | Behavior |
|------|---------|----------|
| Bottom Navigation | Tap "Tasks" tab | Direct navigation |
| Today Dashboard | Tap "See all tasks" | Scrolls to top |
| Push Notification | Tap task notification | Opens list, scrolls to task |
| Quick Capture | After saving task | Returns to list with new task highlighted |
| App Launch | If last screen was Tasks | State restored |

## Interactions

### Tap Interactions

| Target | Action | Result |
|--------|--------|--------|
| Task Card | Single tap | Opens Task Detail bottom sheet |
| Checkbox | Single tap | Completes task with animation |
| Quadrant Badge | Single tap | Opens quadrant quick-selector |
| Goal indicator | Single tap | Navigates to linked goal |
| Section header | Single tap | Toggles collapse/expand |
| FAB | Single tap | Opens Quick Capture flow |
| Search icon | Single tap | Opens search overlay |
| Filter icon | Single tap | Opens filter bottom sheet |

### Swipe Gestures

| Gesture | Action | Visual Feedback |
|---------|--------|-----------------|
| Swipe Right (task) | Complete task | Green background reveals, checkmark icon |
| Swipe Left (task) | Delete task | Red background reveals, trash icon |
| Swipe left (section) | N/A | No action |
| Pull down (list) | Refresh AI classifications | Standard pull-to-refresh indicator |

**Swipe Threshold**: 25% of card width triggers action.

**Swipe Animation**:
1. Background color reveals as card moves
2. Icon fades in at 15% threshold
3. At 25%, haptic feedback
4. Release triggers action with exit animation

### Long Press

| Target | Action | Result |
|--------|--------|--------|
| Task Card | Long press (300ms) | Enters multi-select mode + haptic |
| Task Card (selected) | Long press | Drag to reorder within quadrant |

### Drag and Drop

**Within Quadrant**:
- Long press â†’ lift animation (elevation increases to 8dp)
- Drag to reorder
- Drop position saved immediately
- Other cards animate to make space

**Between Quadrants**:
- Drag task over different quadrant section
- Section highlights when droppable
- Drop changes task's quadrant (manual override)
- Shows "Quadrant changed" snackbar with undo
